### Assignment Contents

This assignment includes the following components:

- Comparison between low-contrast and high-contrast images  
- Histogram analysis of grayscale images  
- Log transformation using different values of the constant *c*  
- Gamma transformation using different gamma (γ) values  
- Contrast stretching using a piecewise linear function with two different breakpoints  
- Intensity level slicing with and without background  
- Decomposition of images into 8 bit-planes  
- Image reconstruction after bit-plane decomposition and Mean Squared Error (MSE) calculation  
- Observations and inferences based on the results
